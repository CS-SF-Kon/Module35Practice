# Module35Practice - Социальная сеть, итоговое практическое задание

Список функций для реализации:
* Регистрация
* Авторизация
* Главная страница с фото и именем
* Изменение данных пользователя
* Добавление друзей
* Отправка сообщений

Необходимо:
1. Создать проект по шаблону MVC - *выполнено*
1. Создать приветствувенную страницу + дизайн - *выполнено*
1. Создать главную страницу (пока пустую)
1. Добавить возможность регистрации, авторизации и выхода:
	1. Подключить БД (EFCore) - *выполнено*
	1. Подключить Identity - *выполнено*
	1. Создать модель пользователя - *выполнено*
	1. Добавить на приветственную страницу блок с информацией о Соцсети, блок регистрации и блок авторизации - *выполнено*
	1. Создать страницу с продолжением регистрации (более полные данные) - *выполнено*
1. Доработать главную страницу:
	1. Должны отображаться имя, фамилия, дата рождения, фото
	1. Реализовать редактирование и сохранение данных пользователя
	1. Реализовать поиск по пользователям и добавление/удаление друзей
1. Реализовать возможность обмена сообщениями:
	1. Проработать модель и репозиторий сообшений
	1. Создать представление чата
1. Тестирование
1. Добавление асинхронности
